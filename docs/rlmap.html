Types

TDirection = (...);

The direction a player can move in.

Values
  • NORTH: walk in direction north
  • EAST: walk in direction east
  • SOUTH: walk in direction south
  • WEST: walk in direction west
TRoomConnections = array[TDirection] of integer;

Array representing the directions in which the player can leave a room. Each field contains an id which room the player then should enter. If there is no room on the connection, it stores -1.

TFields = array[0..2, 0..2] of TField;

2D-array representing the fields in a room.

TRoomList = specialize TFPGList<Troom>;
